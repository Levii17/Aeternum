# Aeternum - Roman Engineering

AeternalIS is a web application built with Next.js and TypeScript, showcasing a portfolio of web design and system architecture projects. The name AeternalIS is a play on the Latin word "aeternus," meaning "eternal," and "IS" for "Information Systems," reflecting the company's philosophy of building enduring digital solutions.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You will need to have Node.js and npm installed on your machine.

```
node -v
npm -v
```

### Installing

1.  Clone the repo
    ```sh
    git clone https://github.com/your_username_/AeternalIS.git
    ```
2.  Install NPM packages
    ```sh
    npm install
    ```
3.  Run the development server
    ```sh
    npm run dev
    ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Running the tests

To run the tests, run the following command:

```sh
npm run test
```
